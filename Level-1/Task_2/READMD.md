# Task 2 - Palindrome Checker

## Objective

Develop a Java program that checks whether a given word or phrase is a palindrome by ignoring spaces, punctuation, and letter case.

## Technologies Used

- Core Java
- Scanner Class
- String Manipulation
- Loops
- Conditional Statements

## Features

- Accepts a word or phrase from the user.
- Ignores spaces and punctuation.
- Converts the input to lowercase for case-insensitive comparison.
- Checks whether the input is a palindrome.
- Displays the result.

## Sample Input

Madam, I'm Adam

## Sample Output

It is a Palindrome.
