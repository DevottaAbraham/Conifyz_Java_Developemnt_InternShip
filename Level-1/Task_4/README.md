# Task 4 - Random Password Generator

## Objective

Develop a Java program that generates a random password based on the user's preferences. The user can specify the password length and choose whether to include uppercase letters, lowercase letters, numbers, and special characters.

## Features

- Accepts the desired password length.
- Allows users to include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Generates a secure random password.
- Uses the `Random` class for random character selection.
- Builds the password efficiently using `StringBuilder`.

## Technologies Used

- Core Java
- Scanner Class
- Random Class
- StringBuilder
- Conditional Statements
- Loops
- Object-Oriented Programming (OOP)

## Sample Output

```text
Enter the length of the Password:
12

Include Uppercase Letters? (Y/N): Y
Include LowerCase Letters? (Y/N): Y
Include Numbers? (Y/N): Y
Include Special Characters? (Y/N): Y

Generated Password:
A7@mQ2#xLp9!
```
