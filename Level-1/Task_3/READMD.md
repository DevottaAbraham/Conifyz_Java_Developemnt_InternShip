# Task 3 - Student Grade Calculator

## Objective

Develop a Java program that calculates the total marks, average marks, and final grade of a student based on the marks entered by the user.

## Features

- Accepts student name and student ID.
- Accepts the number of subjects.
- Stores marks in an integer array.
- Calculates the total marks.
- Calculates the average marks.
- Determines the final grade.
- Validates that marks are between 0 and 100.
- Displays a student report card.

## Technologies Used

- Core Java
- Scanner Class
- Arrays
- Loops
- Conditional Statements
- Object-Oriented Programming (OOP)

## Grade Criteria

| Average Marks | Grade |
|--------------:|:------|
| 90 - 100 | O |
| 80 - 89 | A |
| 70 - 79 | B |
| 60 - 69 | C |
| 50 - 59 | D |
| Below 50 | F |
